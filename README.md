# PopCornCinema

## Про додаток
Сервіс кінотеатру для бронювання, перегляду актуальних новинок у кіно
за допомогою Django фреймворку 

## Особливості
- Адмін панель
- Кошик через сессію без аутентифікації

## У процесі
- створення заказу
- завантаження квитків у pdf файл зі штрихкодом

## Технології
- Python 3.9 
- Django 4.2.8
- Bootstrap 
- Jquery
- AJAX
- SQLite3
